# online-shopping-cart
The Online Shopping Cart project is a web-based application designed to simulate an e-commerce platform where users can browse products, add items to a cart, and manage their purchases efficiently. This project demonstrates the basic functionality of an online shopping system, making it useful for learning full-stack development concepts.A simple and efficient Online Shopping Cart Application built using modern web technologies. This project allows users to browse products, add items to cart, and manage their shopping experience.

📌 Features
🛍️ View available products
➕ Add items to cart
➖ Remove items from cart
🔄 Update product quantity
💰 Calculate total price
📱 Responsive design (works on mobile & desktop)
🛠️ Technologies Used
Frontend: HTML, CSS, JavaScript
Backend (if used): Java / Node.js / PHP (update based on your project)
Database (if used): MySQL / MongoDB
